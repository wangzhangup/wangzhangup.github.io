Build OpenCV with CUDA support

error: nvcc fatal : Unsupported gpu architecture ‘compute_11’

solution: cmake -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/usr/local -D CUDA_GENERATION=Kepler ..
